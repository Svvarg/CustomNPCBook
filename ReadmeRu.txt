nsay name word1 word2 ... wornN - глобальное сообщение во все миры от имени нпс

cb = custombook
cb status состояние маппинга меты книг - существующие диалоги
cb reload - пересоздать маппинг метадаты книг - существующие категории диалогов(Нужно при добавлении новых категорий для новых книг).
cb display - редактирование имени, лора, цвета предмета
cb quest-tag - добавление\удаление к предмету метки делающей его уникальным с невозможностью получить через обычный игровой доступ
cb jump-to-dim - для дебага перемещение в указанное измерение
cb convert - получить простой текст из большой книги библиокрафта
cb dialog - набор инструментов упрощающих создание структур диалогов. подкоманды
сb dialog #id satus - данные о диологе по его id
cb dialog #id text - вывести текст диалога в консоль
cb dialog #id gui - быстрое открытие гуи указанного диалога (пока только гуишку категории в которой храниться диалог)
cb dialog #id options - работа с автогенерацией и удобным редактированием "опций-диалога" - вариантов ответов
cd dialog #id script - взаимодействие с внутренними библиотеками скриптов через содержимое диалога. т.к в самом кастоме нет возможности загружать скрипты так же как и диалог и соотвествующие файлы, чтобы исключить дублирование и хранение их в nbt существ.
